List References

I need to make a list of my sleep cycle, so I figure I might as well just start here.

  1. Wake up at 5 am
  2. Go to bed at 8:40 am
  3. Wake up at 9:00 am
  4. GET THIS CYCLE WORKING

The list above was an ordered list. Below, I will now create an unordered list. The primary difference between these two lists is how they start off. As we can see, an ordered list (ol) uses numbers to split. An unordered list (ul) just uses bullet points.

  • For step 2, gotta sleep at 2:20 pm
  • Wake up at 2:40 pm
  • Last sleep at 7:40 pm
  • Finish off waking up at 8:00 pm
