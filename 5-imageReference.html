3 rules to images

  1. Save images in the right format
    • Mainly use jpeg, gif, or png for images. If not, speed of loading could be affected
  2. Save images at the right size
    • Save images at the same size you wish it to be displayed on the website. It's just easier to do this way! Don't resize normally
  3. Measure Images in pixels
    • When saving pictures, save them in pixels so that they can be compared from monitor to monitor!
