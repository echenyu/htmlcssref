Let's Learn Tables!

Obviously, a table starts out with the table tag. Elements are represented with (td) and row/column headings are represented with (th). Use the scope attribute to see if the heading is for a row or column.

The colspan and rowspan attributes make it possible to let the (td) go across more than just one column or one row. In cycle 3, "No data available" spans two columns

Whenever creating tables, distinguish the main content from the top and bottom by using the (thead), (tbody), and (tfoot) tags. These tags will help create separation. Primarily useful for larger tables though!
