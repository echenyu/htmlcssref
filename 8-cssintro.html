Intro to CSS

To start with CSS, we need to include a link to the CSS file within our directory. To do so, we just use the (link) element. The type attribute is always "text/css," and the rel (relationship) is always "stylesheet."

In general, we have a selector and a declaration. The selector helps us choose which part of the text/"box" to apply the CSS stylesheet to. Declarations help determine what exactly to change! This paragraph changed with an id.
