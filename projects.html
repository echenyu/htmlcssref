EECS 281 Projects

Project 1: Harry Potter

For this project, we were asked to simulate both a stack and queue based path finding program. Harry potter was in a map of N x N size, and we had to navigate the map in the quickest way possible to find the Ring.
Data Structures Used: Stacks and Queue.

Project 2: Mine Escape

For this project, we were asked to implement priority queues using a Poorman method, a sorted method, a binary heap method, and a pairing heap method. This project required a lot of referring to documentation.
Data Structures Used: Priority Queues

Project 3: Stock Market Simulator

For this project, we were asked to create a program that could simulate a stock market based on a list of orders that came in and out.
Data Structures Used: Priority Queues, Unordered Maps, Unordered Sets

Project 4: Amazon Drone Delivery

For this project, we were asked to create a program that would simulate the path taken by an amazon drone based on the available edges and vertices. Created implementations for Prim's and Kruskal's algorithms. In addition, there was also a Traveling Salesman problem.
Data Structures Used: Priority Queue, Vectors
